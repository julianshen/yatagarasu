## Learning Path

### For Developers

1. Start with [Basic Proxy Setup](/yatagarasu/tutorials/basic-proxy/)
2. Add [JWT Authentication](/yatagarasu/tutorials/jwt-authentication/)
3. Enable [Caching](/yatagarasu/tutorials/caching/) for better performance

### For DevOps/SRE

1. Review [Kubernetes Quickstart](/yatagarasu/getting-started/kubernetes/)
2. Set up [High Availability](/yatagarasu/tutorials/high-availability/)
3. Configure [Production Monitoring](/yatagarasu/tutorials/monitoring/)

### For Security Engineers

1. Start with [JWT Authentication](/yatagarasu/tutorials/jwt-authentication/)
2. Explore [OPA Authorization](/yatagarasu/tutorials/opa-authorization/)
3. Implement [OpenFGA](/yatagarasu/tutorials/openfga/) for fine-grained control
