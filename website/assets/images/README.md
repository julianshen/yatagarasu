# Logo Assets

Place the following files in this directory:

1. **logo.png** - The main Yatagarasu logo (the three-legged crow with red sun and Japanese calligraphy)
2. **favicon.ico** - Optional favicon for the website

The logo should be the image with:
- Black three-legged crow (八咫烏)
- Red sun behind
- Brush stroke circle (enso)
- Japanese characters "八咫烏"
- "YATAGARASU" text below
